# SetIn - util function

An utility function for setting deep object value by giving the following 5 arguments: object, path1, path2, path3, value

## Technologies

Project is created with:

- node
- nodemon
- mocha
- chai

## Setup

To run this project, do the following (node required):

```
$ npm install
$ npm run start
```

## Test

```
$ npm run test
```

## Local dev

```
$ npm run start:dev
```
